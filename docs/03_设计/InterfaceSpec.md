# 接口说明

## CLI 命令
- list/get/update/append-log/suggest-branch/current
- list/get 会自动补齐表头

## tasks.xlsx 字段（中文表头）
- TaskID：任务ID
- Title：标题
- Module：模块
- Priority：优先级
- Status：状态
- Owner：负责人
- Estimate：预估工时
- DependsOn：依赖
- Acceptance：验收标准
- Notes：备注

## 兼容字段
- REQ_ID/Branch/Progress/LastUpdate/WorkLog/MR_URL
